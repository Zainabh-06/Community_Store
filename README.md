# Community Store

A MERN Stack e-commerce application that allows users to browse products, manage carts, place orders, and securely authenticate using JWT in their community.

## Features

* User Registration & Login
* JWT Authentication
* Product Management
* Shopping Cart
* Order Management
* Protected Routes

## Tech Stack

* **Frontend:** React, Vite
* **Backend:** Node.js, Express.js
* **Database:** MongoDB, Mongoose
* **Authentication:** JWT, Bcrypt
